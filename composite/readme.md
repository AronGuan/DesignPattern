组合模式   
定义:允许你将对象组合成树形结构来表现"整体/部分"层次结构，组合让客户以一致的方式处理个别对象以及对象组合。(即能够对整个结构应用相同的操作(例如打印))，枝和叶可以有一样的操作

最上层的根定义接口，叶是基本的POJO类，而枝实现了操作方法。
